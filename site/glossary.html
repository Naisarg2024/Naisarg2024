Glossary

HTML
HyperText Markup Language, the standard language used to create and structure webpages.
CSS
Cascading Style Sheets, used to style HTML elements and control the layout of webpages.
JavaScript
A programming language that adds interactivity, dynamic content, and functionality to webpages.
Git
A version control system that helps track changes in code, allowing collaboration and history tracking.
VS Code
A powerful and lightweight code editor developed by Microsoft, widely used for web development.
